Spaceship Illustration


This project required me to design 3 artboards as a backgroudnd for a 3D object in Adobe Aero.



Information


This illustration was done with Adobe Illustrator with 3 different artboards joined together. The goal of the project is to place all
3 art boards into Adobe Aero, together with a 3D character which matches the theme of the background created.
All illustrations were made from scratch in Adobe Illustrator.

Tools Used In Project
