Information


Done in Year 2 of my polytechnic journey, this project required us to recrate or revamp an app. I chose to revamp the TraceTogther app.
We had to conduct competitive analysis, introduce new features to be implemented and eventually come up with both low and high fidellity prototypes of these newly implemented features on both Apple Watch & Mobile screens.
We were also required to conduct user acceptance testings on the designed prototypes, with our chosen user persona.
All graphics and flows in the wireframes are done from scratch.

Tools Used In Project


Figma, Google (Research), Canva, Microsoft Word
