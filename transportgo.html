Transport GO (Prototype) UI/UX Project


This project done in Year 1 of my polytechnic journey required me to create an app with the intention of encouraging users to take public transport. This project showcases a reward based game system & hand crafted UI/UX designs.


Click to view demo video

Information


This project required me to create a game flow showcasing a reward system which encourages users to take more public transport.
The game features a progress map and inventory where users can buy items to customse their characters with or earn free rides by playing the in app game or reaching a certain level of the progress map.
The game also has a share feature which users can post their progress updates on to their social media platforms.
